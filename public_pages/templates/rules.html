Below is the set of rules for a correct index file. This application will do whatever it can to make your file match these criteria.

  1. There should be no white space at the beginning and at the end of the file.
  2. There should be an empty line before every new index letter.
  3. First tier of entries does not have spaces nor dashes at the beginning.
  4. Every next tier have a new solid dash and a space before the entry.
  5. Tiers can only "grow" per one tier at the time.
  6. Entries on every tier are sorted within their own group.
  7. Sorting is done alphabetically according to Polish dictionary.
